# Condiciones de Servicio de StarzzBot

**Última actualización:** 25 de junio de 2026

Al invitar a **StarzzBot** a tu servidor de Discord o interactuar con él, aceptas cumplir con estos Términos de Servicio y con las [Directivas de la Comunidad de Discord](https://discord.com). Si no estás de acuerdo, por favor elimina el bot de tu servidor.

---

## 1. Propósito del Bot
**StarzzBot** es un bot personalizado para la comunidad de nuestra creadora de contenido de YouTube. Su función principal es el entretenimiento, la moderación básica, la entrega de roles y la automatización de notificaciones para los miembros del servidor.

## 2. Datos y Privacidad
Para funcionar correctamente, **StarzzBot** procesa datos públicos proporcionados por la API de Discord:
* **Identificadores (IDs):** Guardamos IDs de usuario y de servidor para aplicar configuraciones personalizadas y roles.
* **Mensajes:** El bot solo lee comandos directos del usuario para responder a sus solicitudes.
* **Datos Sensibles:** El bot **no** recopila, almacena ni comparte contraseñas, correos electrónicos ni datos bancarios.

## 3. Uso Aceptable
Como usuario del bot, te comprometes a:
* No intentar explotar errores (*bugs*) o vulnerabilidades del bot.
* No usar los comandos del bot para acosar, hacer spam o molestar a otros miembros del servidor.
* Respetar las reglas generales del servidor de la comunidad de YouTube.

## 4. Limitación de Responsabilidad
**MamaBot** se ofrece de forma gratuita y "tal cual". No garantizamos que el servicio funcione de manera ininterrumpida. El equipo de desarrollo no se hace responsable por caídas del servicio, pérdida de configuraciones temporales o sanciones aplicadas por moderadores mediante el uso del bot.

## 5. Cambios en los Términos
Nos reservamos el derecho de modificar estas condiciones en cualquier momento. Los cambios importantes se notificarán dentro del servidor de Discord de la comunidad.

## 6. Soporte y Contacto
Si encuentras un error o tienes dudas sobre el funcionamiento de **MamaBot**, por favor contacta al equipo de moderación o abre un reporte en los canales correspondientes de nuestro servidor de Discord.
